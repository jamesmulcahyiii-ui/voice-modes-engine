# Voice Modes Engine

Voice Modes Engine is a modular prompt-and-content system that transforms raw ideas, brain dumps, transcripts, chats, and outlines into finished outputs across different speech modes while preserving a consistent personal voice.

## Core Concept

Raw thought should not stay trapped in brainstorm form. This project converts raw input into deployable content through five dials:

1. Core voice
2. Speech mode
3. Formality level
4. Purpose
5. Output container

## Supported Modes

- Legal writing
- Academic writing
- Creative writing
- Blog writing
- Social media
- Sales and offer writing
- Technical documentation
- App/product specification
- Newsletter
- Personal essay
- Blunt accountability
- Spiritual/philosophical writing
- Video scripts
- Prompt engineering

## Workflow

1. Capture raw idea
2. Parse the idea
3. Select speech mode
4. Set formality/audience/purpose
5. Generate draft
6. Repurpose
7. Deploy
